fiftyone.core.collections.SaveContext

class documentation

class SaveContext(object):

Constructor: SaveContext(sample_collection, batch_size, batching_strategy)

View In Hierarchy

Context that saves samples from a collection according to a configurable batching strategy. Args: sample_collection: a :class:`fiftyone.core.collections.SampleCollection` batch_size (None): the batch size to use. If a ``batching_strategy`` is provided, this parameter configures the strategy as described below. If no ``batching_strategy`` is provided, this can either be an integer specifying the number of samples to save in a batch (in which case ``batching_strategy`` is implicitly set to ``"static"``) or a float number of seconds between batched saves (in which case ``batching_strategy`` is implicitly set to ``"latency"``) batching_strategy (None): the batching strategy to use for each save operation. Supported values are: - ``"static"``: a fixed sample batch size for each save - ``"size"``: a target batch size, in bytes, for each save - ``"latency"``: a target latency, in seconds, between saves By default, ``fo.config.default_batcher`` is used
