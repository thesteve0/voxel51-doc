def locate_annotation_config(): ¶

Returns the path to the :class:`AnnotationConfig` on disk. The default location is ``~/.fiftyone/annotation_config.json``, but you can override this path by setting the ``FIFTYONE_ANNOTATION_CONFIG_PATH`` environment variable. Note that a config file may not actually exist on disk. Returns: the path to the :class:`AnnotationConfig` on disk

def locate_app_config(): ¶

Returns the path to the :class:`AppConfig` on disk.
