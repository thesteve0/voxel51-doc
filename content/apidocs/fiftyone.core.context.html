def _get_context(): ¶

Determine the most specific context that we're in. Returns: one of - ``_COLAB``: we're in Colab with an IPython notebook context - ``_DATABRICKS``: we're in Databricks with an IPython notebook context - ``_IPYTHON``: we're in an IPython notebook context (e.g., from running `jupyter notebook` at the command line) - ``_NONE``: we're in a non-notebook context, e.g., a Python script or a Python REPL
