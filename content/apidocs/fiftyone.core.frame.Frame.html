class Frame(Document):

Constructor: Frame(**kwargs)

View In Hierarchy

A frame in a video :class:`fiftyone.core.sample.Sample`. Frames store all information associated with a particular frame of a video, including one or more sets of labels (ground truth, user-provided, or FiftyOne-generated) as well as additional features associated with subsets of the data and/or label sets. .. note:: :class:`Frame` instances that are attached to samples **in datasets** are singletons, i.e., ``sample.frames[frame_number]`` will always return the same :class:`Frame` instance.
