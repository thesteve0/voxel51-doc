class GeoLocations(_HasID, Label):

View In Hierarchy

A batch of location data in GeoJSON format. The attributes of this class accept lists of data in the format of the corresponding attributes of :class:`GeoLocation`. Args: points (None): a list of points lines (None): a list of lines polygons (None): a list of polygons
