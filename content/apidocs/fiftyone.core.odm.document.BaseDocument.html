class BaseDocument(MongoEngineBaseDocument):

Known subclasses: fiftyone.core.odm.document.Document, fiftyone.core.odm.document.DynamicDocument

View In Hierarchy

Base class for documents that are written to the database in their own collections. The ID of a document is automatically populated when it is added to the database, and the ID of a document is ``None`` if it has not been added to the database. Attributes: id: the ID of the document, or ``None`` if it has not been added to the database
