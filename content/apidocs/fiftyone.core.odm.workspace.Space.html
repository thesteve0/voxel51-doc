class Space(AppComponent):

Constructor: Space(*args, **kwargs)

View In Hierarchy

Document for configuration of a Space in the App. Args: component_id: the component's ID children: the list of :class:`Component` children of this space, if any orientation (["horizontal", "vertical"]): the orientation of this space's children active_child: the ``component_id`` of this space's currently active child sizes: the ordered list of relative sizes for children of a space in ``[0, 1]``
