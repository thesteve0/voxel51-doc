class Flatten(ViewStage):

Constructor: Flatten(stages)

View In Hierarchy

Returns a flattened view that contains all samples in a dynamic grouped collection. Examples:: import fiftyone as fo import fiftyone.zoo as foz from fiftyone import ViewField as F dataset = foz.load_zoo_dataset("cifar10", split="test") # Group samples by ground truth label grouped_view = dataset.take(1000).group_by("ground_truth.label") print(len(grouped_view)) # 10 # Return a flat view that contains 10 samples from each class stage = fo.Flatten(fo.Limit(10)) flat_view = grouped_view.add_stage(stage) print(len(flat_view)) # 100 Args: stages (None): a :class:`ViewStage` or list of :class:`ViewStage` instances to apply to each group's samples while flattening
