def extract_archive(archive_path, outdir=None, cleanup=False): ¶

Extracts the contents of an archive. The following formats are guaranteed to work: ``.zip``, ``.tar``, ``.tar.gz``, ``.tgz``, ``.tar.bz``, ``.tbz``. If an archive *not* in the above list is found, extraction will be attempted via the ``patool`` package, which supports many formats but may require that additional system packages be installed. Args: archive_path: the archive path outdir (None): the directory into which to extract the archive. By default, the directory containing the archive is used cleanup (False): whether to delete the archive after extraction
