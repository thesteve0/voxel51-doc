fiftyone.core.threed.lights.PointLight

class documentation

class PointLight(Light):

Constructor: PointLight(name, distance, decay, color, ...)

View In Hierarchy

Represents a point light. Args: name ("PointLight"): the name of the light distance (0.0): the distance at which the light's intensity is zero decay (2.0): the amount the light dims along the distance of the light color ("#ffffff"): the color of the light intensity (1.0): the intensity of the light in the range ``[0, 1]`` visible (True): default visibility of the object in the scene position (None): the position of the light in object space quaternion (None): the quaternion of the light in object space scale (None): the scale of the light in object space
