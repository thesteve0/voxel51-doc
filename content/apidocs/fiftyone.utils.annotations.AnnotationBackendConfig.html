Base class for configuring an :class:`AnnotationBackend` instances. Subclasses are free to define additional keyword arguments if they desire. Args: name: the name of the backend label_schema: a dictionary containing the description of label fields, classes and attributes to annotate media_field ("filepath"): string field name containing the paths to media files on disk to upload **kwargs: any leftover keyword arguments after subclasses have done their parsing
