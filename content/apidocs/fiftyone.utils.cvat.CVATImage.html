class CVATImage(object):

Constructor: CVATImage(id, name, width, height, ...)

View In Hierarchy

An annotated image in CVAT image format. Args: id: the ID of the image name: the filename of the image width: the width of the image, in pixels height: the height of the image, in pixels tags (None): a list of :class:`CVATImageTag` instances boxes (None): a list of :class:`CVATImageBox` instances polygons (None): a list of :class:`CVATImagePolygon` instances polylines (None): a list of :class:`CVATImagePolyline` instances points (None): a list of :class:`CVATImagePoints` instances subset (None): the project subset of the image, if any
