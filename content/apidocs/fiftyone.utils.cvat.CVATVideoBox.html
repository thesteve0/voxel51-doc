class CVATVideoBox(CVATVideoAnno):

Constructor: CVATVideoBox(frame, label, xtl, ytl, ...)

View In Hierarchy

An object bounding box in CVAT video format. Args: frame: the 0-based frame number label: the object label string xtl: the top-left x-coordinate of the box, in pixels ytl: the top-left y-coordinate of the box, in pixels xbr: the bottom-right x-coordinate of the box, in pixels ybr: the bottom-right y-coordinate of the box, in pixels outside (None): whether the object is outside (invisible) occluded (None): whether the object is occluded keyframe (None): whether the frame is a keyframe attributes (None): a list of :class:`CVATAttribute` instances
