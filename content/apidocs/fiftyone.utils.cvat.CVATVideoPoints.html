class CVATVideoPoints(CVATVideoAnno, HasCVATPoints):

Constructor: CVATVideoPoints(frame, label, points, outside, ...)

View In Hierarchy

A set of keypoints in CVAT video format. Args: frame: the 0-based frame number label: the keypoints label string points: a list of ``(x, y)`` pixel coordinates defining the keypoints outside (None): whether the keypoints is outside (invisible) occluded (None): whether the keypoints are occluded keyframe (None): whether the frame is a keyframe attributes (None): a list of :class:`CVATAttribute` instances
