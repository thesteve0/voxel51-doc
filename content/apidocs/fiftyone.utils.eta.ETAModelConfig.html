class ETAModelConfig(fom.ModelConfig):

View In Hierarchy

Meta-config class that encapsulates the configuration of an `eta.core.learning.Model` that is to be run via the :class:`ETAModel` wrapper. Example:: import fiftyone.core.models as fom model = fom.load_model({ "type": "fiftyone.utils.eta.ETAModel", "config": { "type": "eta.detectors.YOLODetector", "config": { "model_name": "yolo-v2-coco" } } }) Args: type: the fully-qualified class name of the :class:`fiftyone.core.models.Model` subclass, which must be :class:`ETAModel` or a subclass of it config: a dict containing the ``eta.core.learning.ModelConfig`` for the ETA model
