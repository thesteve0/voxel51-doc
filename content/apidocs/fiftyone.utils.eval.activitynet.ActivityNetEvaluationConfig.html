class ActivityNetEvaluationConfig(DetectionEvaluationConfig):

Constructor: ActivityNetEvaluationConfig(pred_field, gt_field, iou, classwise, ...)

View In Hierarchy

ActivityNet-style evaluation config. Args: pred_field: the name of the field containing the predicted :class:`fiftyone.core.labels.TemporalDetection` instances gt_field: the name of the field containing the ground truth :class:`fiftyone.core.labels.TemporalDetection` instances iou (None): the IoU threshold to use to determine matches classwise (None): whether to only match segments with the same class label (True) or allow matches between classes (False) compute_mAP (False): whether to perform the necessary computations so that mAP and PR curves can be generated iou_threshs (None): a list of IoU thresholds to use when computing mAP and PR curves. Only applicable when ``compute_mAP`` is True
