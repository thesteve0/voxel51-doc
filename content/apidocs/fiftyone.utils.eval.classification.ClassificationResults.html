class ClassificationResults(BaseEvaluationResults):

Known subclasses: fiftyone.utils.eval.classification.BinaryClassificationResults

Constructor: ClassificationResults(samples, config, eval_key, ytrue, ...)

View In Hierarchy

Class that stores the results of a classification evaluation. Args: samples: the :class:`fiftyone.core.collections.SampleCollection` used config: the :class:`ClassificationEvaluationConfig` used eval_key: the evaluation key ytrue: a list of ground truth labels ypred: a list of predicted labels confs (None): an optional list of confidences for the predictions weights (None): an optional list of sample weights ytrue_ids (None): a list of IDs for the ground truth labels ypred_ids (None): a list of IDs for the predicted labels classes (None): the list of possible classes. If not provided, the observed ground truth/predicted labels are used missing (None): a missing label string. Any None-valued labels are given this label for evaluation purposes backend (None): a :class:`ClassificationEvaluation` backend
