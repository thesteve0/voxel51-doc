class SimpleEvaluationConfig(RegressionEvaluationConfig):

Constructor: SimpleEvaluationConfig(pred_field, gt_field, metric, **kwargs)

View In Hierarchy

Base class for configuring :class:`SimpleEvaluation` instances. Args: pred_field: the name of the field containing the predicted :class:`fiftyone.core.labels.Regression` instances gt_field: the name of the field containing the ground truth :class:`fiftyone.core.labels.Regression` instances metric ("squared_error"): the error metric to use to populate sample/frame-level error data. Supported values are ``("squared_error", "absolute_error")`` or any function that accepts two scalar arguments ``(ypred, ytrue)``
