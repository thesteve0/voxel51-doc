class GitHubRepository(object):

Constructor: GitHubRepository(repo)

View In Hierarchy

Class for interacting with a GitHub repository. .. note:: To interact with private GitHub repositories that you have access to, provide your GitHub personal access token by setting the ``GITHUB_TOKEN`` environment variable. Args: repo: the GitHub repository or identifier, which can be: - a GitHub repo URL like ``https://github.com/<user>/<repo>`` - a GitHub ref like ``https://github.com/<user>/<repo>/tree/<branch>`` or ``https://github.com/<user>/<repo>/commit/<commit>`` - a GitHub ref string like ``<user>/<repo>[/<ref>]``
