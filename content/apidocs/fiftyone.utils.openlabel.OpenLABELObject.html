class OpenLABELObject(AttributeParser):

Constructor: OpenLABELObject(key, name, type, bboxes, ...)

View In Hierarchy

An object parsed from OpenLABEL labels. Args: key: the OpenLABEL key string for this object name (None): the name string of the object type (None): the type string of the object bboxes (None): an :class`OpenLABELShapes` of bounding boxes for this object segmentations (None): an :class`OpenLABELShapes` of polygon segmentations for this object keyponts (None): an :class:`OpenLABELShapes` of keypoints for this object stream (None): the :class:`OpenLABELStream` this object corresponds to other_attrs (None): a dict of attributes and their values for this object is_frame_level (False): whether this object is sample-level or frame-level
