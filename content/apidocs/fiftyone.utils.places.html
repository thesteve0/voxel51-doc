def download_places_dataset_split(dataset_dir, split, raw_dir=None): ¶

Utility that downloads splits of the `Places dataset <http://places2.csail.mit.edu/index.html>`. Any existing files are not re-downloaded. Args: dataset_dir: the directory to download the dataset split: the split to download. Supported values are ``("train", "validation", "test")`` raw_dir (None): a directory in which full annotations files may be stored to avoid re-downloads in the future Returns: a tuple of: - num_samples: the total number of downloaded images - classes: the list of all classes - did_download: whether any content was downloaded (True) or if all necessary files were already downloaded (False)
