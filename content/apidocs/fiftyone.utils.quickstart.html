def quickstart(video=False, port=None, address=None, remote=False): ¶

Runs the FiftyOne quickstart. This method loads an interesting dataset from the Dataset Zoo, launches the App, and prints some suggestions for exploring the dataset. Args: video (False): whether to launch a video dataset port (None): the port number to serve the App. If None, ``fiftyone.config.default_app_port`` is used address (None): the address to serve the App. If None, ``fiftyone.config.default_app_address`` is used remote (False): whether this is a remote session, and opening the App should not be attempted Returns: a tuple containing - dataset: the :class:`fiftyone.core.dataset.Dataset` that was loaded - session: the :class:`fiftyone.core.session.Session` instance for the App that was launched
