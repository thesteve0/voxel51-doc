class TorchEmbeddingsMixin(fom.EmbeddingsMixin):

Known subclasses: fiftyone.utils.torch.TorchImageModel

Constructor: TorchEmbeddingsMixin(model, layer_name, as_feature_extractor)

View In Hierarchy

Mixin for Torch models that can generate embeddings. Args: model: the Torch model, a :class:`torch:torch.nn.Module` layer_name (None): the name of the embeddings layer whose output to save, or ``None`` if this model instance should not expose embeddings. Prepend ``"<"`` to save the input tensor instead as_feature_extractor (False): whether to operate the model as a feature extractor. If ``layer_name`` is provided, this layer is passed to torchvision's ``create_feature_extractor()`` function. If no ``layer_name`` is provided, the model's output is used as-is for feature extraction
