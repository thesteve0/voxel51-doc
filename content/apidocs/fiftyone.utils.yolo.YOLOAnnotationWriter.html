Writes the labels to disk. Args: label: a :class:`fiftyone.core.labels.Detections` or :class:`fiftyone.core.labels.Polylines` txt_path: the path to write the annotation TXT file labels_map_rev: a dictionary mapping class label strings to target integers dynamic_classes (False): whether to dynamically add new labels to ``labels_map_rev`` include_confidence (False): whether to include confidences in the export, if they exist
